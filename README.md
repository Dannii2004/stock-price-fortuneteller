#stock-price-fortuneteller

## Descriptrion
 Uses linear regression to predict the price for a specifc stock and timeframe. Predicts future stock prices based on old price data.

##Prereqs
Before running the script, make sure you have:

- Python 3
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
